Quantum Mind: Early Detection of Mental Health Crises Using
Quantum Machine Learning

Spatiotemporal-Aware Student Depression Screening via Variational Quantum Classifiers, Voice Biometrics & Neurocognitive Gaming
Arjun  ·  PennyLane · 14-Qubit VQC · Hume EVI · Meta-Llama 3.1 · React · Flask · Google Cloud Run
Abstract

We present Quantum Mind, a novel mental health triage system that leverages a Variational Quantum Classifier (VQC) to detect depression risk and mental health crises in students. By mapping 14 psychological and behavioral features onto a 14-qubit quantum circuit, we exploit quantum entanglement to model complex, non-linear symptom interactions that classical methods struggle to capture.

The system integrates neurocognitive games, real-time voice biometric analysis (Hume EVI), and AI-powered transcript processing (Meta-Llama 3.1) into a holistic, multimodal assessment pipeline deployed on Google Cloud Run with real-time inference.

Introduction & Problem Statement

Depression is a leading cause of disability among students worldwide. Traditional screening relies on self-report surveys alone, missing critical behavioral and cognitive signals. Classical ML models struggle with the non-linear entanglement between psychological symptoms where anxiety, sleep disruption, isolation, and cognitive decline interact in complex feedback loops.

Hypothesis

Quantum machine learning can detect complex, non-linear relationships between psychological symptoms (modeled as quantum entanglement) more effectively than classical approaches, enabling earlier and more accurate mental health risk identification in student populations.

Limitations Addressed
  • Self-report surveys alone miss behavioral & cognitive signals
  • Classical models underfit multi-symptom interaction dynamics
  • No existing multimodal screening tools for students
  • Early intervention requires automated, real-time triage
Research Objectives
  1. Build a VQC-based classifier mapping 14 features to mental health risk probability
  2. Integrate biometric voice analysis for anxiety & isolation detection via Hume EVI
  3. Design neurocognitive games measuring sustained attention & working memory
  4. Achieve clinical-grade risk stratification (Low / Moderate / High / Crisis)
  5. Deploy a production-ready, containerized cloud triage system
  6. Evaluate comprehensively using 15+ statistical metrics
